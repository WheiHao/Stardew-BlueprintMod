# Farm Blueprint

一个用于《星露谷物语》`SMAPI` 的农场蓝图工具。  
你可以把现有农场区域保存成蓝图，也可以导入别人分享的蓝图并在游戏里预览、一键放置。

## 功能简介

- 将当前场景中的一块区域保存为蓝图
- 读取本地蓝图文件并在游戏内浏览
- 预览蓝图摆放效果
- 一键放置蓝图
- 支持撤销上一次一键放置
- 支持覆盖模式与创造模式切换

## 安装方法

1. 安装最新版《星露谷物语》
2. 安装 `SMAPI 4.0.0` 或更高版本
3. 将本 mod 文件夹解压到游戏的 `Mods` 目录中
4. 启动游戏，确认 `Farm Blueprint` 被 `SMAPI` 正常加载

可选：

- 安装 `Generic Mod Config Menu`，这样可以在游戏内修改快捷键和部分设置

## 快速开始

### 使用别人分享的蓝图

1. 将下载到的蓝图 `.json` 文件放进本 mod 的 `blueprints` 文件夹
2. 进入存档后，按 `Ctrl + L` 打开蓝图库
3. 点击想使用的蓝图，进入预览模式
4. 移动到想放置的位置
5. `左键` 放置蓝图，`右键` 退出预览
6. 预览时可用 `← / →` 切换其他蓝图

### 保存自己的蓝图

1. 站在想保存的区域附近
2. 按住 `Ctrl`
3. `左键` 点击第一个角作为起点
4. 继续按住 `Ctrl`，再次 `左键` 点击另一个角作为终点
5. 输入蓝图名称并确认保存

保存完成后，蓝图会自动写入本 mod 的 `blueprints` 文件夹。

## 默认快捷键

- `Ctrl + L`：打开蓝图库 / 进入蓝图预览
- `Ctrl + Z`：撤销上一次一键放置
- `Ctrl + C`：清除所有虚影
- `K`：切换创造模式
- `O`：切换覆盖模式
- `Ctrl + 左键`：框选并保存蓝图
- `预览模式左键`：尝试放置蓝图
- `预览模式右键`：退出预览模式
- `预览模式 ← / →`：切换上一个 / 下一个蓝图

如果你安装了 `Generic Mod Config Menu`，可以在游戏内修改这些按键。

## 蓝图文件位置

蓝图文件存放在本 mod 目录下的：

```text
blueprints/
```

你可以：

- 把别人分享的 `.json` 蓝图放进这个文件夹
- 把自己保存出来的蓝图从这个文件夹里复制出来分享给别人

## 模式说明

### 覆盖模式

- 开启时：放置蓝图时会按当前逻辑覆盖目标位置
- 关闭时：更适合谨慎摆放，避免误覆盖原有内容

### 创造模式

- 开启时：更偏向自由摆放，不依赖正常材料消耗流程
- 默认情况下此模式可在配置中禁用，以避免影响正常游玩平衡

## 建议的分享方式

如果你想把自己的蓝图分享给其他玩家，建议一起提供：

- 蓝图 `.json` 文件
- 一张完成效果截图
- 蓝图适用的地图区域说明
- 是否需要开启覆盖模式 / 创造模式

这样别人会更容易一次放对位置。

## 常见问题

### 为什么打开蓝图库后没有蓝图？

因为 `blueprints` 文件夹里还没有任何 `.json` 蓝图文件。  
请先自己保存一个蓝图，或把别人分享的蓝图文件放进去。

### 为什么不能切换创造模式？

因为创造模式可能在配置中被禁用了。  
如果安装了 `Generic Mod Config Menu`，可以在配置界面中开启。

### 蓝图放置失败怎么办？

先检查：

- 当前位置是否足够摆下整张蓝图
- 是否因为覆盖模式关闭而无法放置
- 是否需要切换到更合适的模式
- 是否满足当前蓝图的材料或放置条件
